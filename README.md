# Deloitte Data Analytics Virtual Experience

## Overview

This project demonstrates data analysis skills using Tableau and Excel as part of Deloitte's virtual experience program.

## Task 1: Telemetry Data Analysis (Tableau)

* Created calculated field "Unhealthy" to measure downtime
* Built dashboards:

  * Down Time per Factory
  * Down Time per Device Type
* Applied filters to analyze factory-level performance
* Identified factory with highest downtime

## Task 2: Equality Analysis (Excel)

* Classified equality scores into:

  * Fair
  * Unfair
  * Highly Discriminative
* Used Excel formulas: IF, AND, OR
* Analyzed fairness across job roles and factories

## Tools Used

* Tableau
* Microsoft Excel
